# Gasty App Technology Stack

## Overview
The Gasty App is built with a modern, full-stack JavaScript/TypeScript approach, leveraging a monorepo structure to manage both frontend and backend components.

## Backend
-   **Language:** TypeScript
-   **Runtime:** Node.js
-   **Framework:** Express.js (for building RESTful APIs)
-   **Database:** MongoDB (using Mongoose for object data modeling)
-   **Architecture:** Hexagonal Architecture (Ports and Adapters pattern)

## Frontend
-   **Language:** TypeScript
-   **Framework:** Next.js (React Framework for production)
-   **Styling:** Tailwind CSS (for utility-first CSS styling)

## Development Tools & Practices
-   **Version Control:** Git
-   **Package Manager:** Yarn/NPM
-   **Testing:** Vitest (for unit and integration testing)
-   **Linting:** ESLint
-   **Code Formatting:** Prettier (implied by typical JS/TS projects)
